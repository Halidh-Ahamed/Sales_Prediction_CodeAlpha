# Sales Prediction - CodeAlpha Internship

This project is part of my CodeAlpha Data Science Internship.

## Project Overview
The goal of this project is to predict product sales based on advertising expenditure using machine learning techniques.

The dataset contains advertising budgets for different media channels and corresponding sales.

## Features
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Workflow
1. Importing the dataset
2. Data exploration and visualization
3. Data preprocessing
4. Training a regression model
5. Evaluating model performance
6. Predicting sales based on advertising budget
